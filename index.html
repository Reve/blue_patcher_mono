The Blue Patcher written in Mono C# so it's compatible with OS X too.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to GitHub Pages.

Usage

Start the blue_patcher.exe (on Windows) or blue_patcher.command (on Mac/Linux) and navigate to the blue.dll location. If it didn't output "success", maybe the patcher is not updated. In this case please send me a PM to modify it or feel free to do so yourself. This program was made for educational purposes, use it at your own risk.
